RepoPeek

A native macOS menu bar app for GitLab repositories. Track pipelines, issues, merge requests, releases, local Git state, activity, and rate limits without keeping a browser tab open.

Read the README Cache Design Account Model Spec

GitLab First

  • GitLab.com and self-managed hosts
  • Full group/subgroup/project paths
  • PAT authentication only

Menu Workflow

  • Repository cards and filters
  • Issues, MRs, pipelines, releases
  • Local branch and worktree state

Operational Detail

  • REST cache and ETags
  • Rate-limit visibility
  • Settings-based diagnostics
